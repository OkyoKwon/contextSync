# 디자인 시스템

ContextSync 프론트엔드 디자인 시스템. Tailwind CSS 4 + CSS custom properties 기반, dark-first 테마, JetBrains Mono 모노스페이스 폰트.

---

## 1. 컬러 토큰

## 2. 타이포그래피

- **폰트:** JetBrains Mono (Google Fonts, weights: 400, 500, 600, 700)
- **적용:** `font-mono` 클래스가 AppLayout 루트에 설정 → 전체 앱에 적용
- **사이즈 스케일:** `text-xs` → `text-sm` → `text-base` → `text-lg` → `text-xl`
- **자주 사용되는 조합:**
  - 본문: `text-sm text-text-secondary`
  - 라벨: `text-sm font-medium text-text-tertiary`
  - 제목: `text-xl font-bold text-text-primary`
  - 보조: `text-xs text-text-muted`

## 4. 컴포넌트 카탈로그

모든 UI 컴포넌트: `apps/web/src/components/ui/`

### Button

```typescript
interface ButtonProps extends ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary' | 'danger' | 'ghost';
  size?: 'sm' | 'md' | 'lg';
}
```

| Variant | 스타일 |
|---------|--------|
| `primary` | btn-primary-bg/text, 테마 반전색 |
| `secondary` | surface-hover 배경, zinc ring |
| `danger` | red-600 배경, red-700 hover |
| `ghost` | 투명, text-tertiary, interactive-hover |

| Size | 클래스 |
|------|--------|
| `sm` | `px-3 py-1.5 text-sm` |
| `md` | `px-4 py-2 text-sm` (기본) |
| `lg` | `px-6 py-3 text-base` |

공통: `inline-flex items-center justify-center rounded-lg font-medium`, disabled 시 `opacity-50 cursor-not-allowed`.

### Card

```typescript
interface CardProps extends HTMLAttributes<HTMLDivElement> {
  padding?: 'none' | 'sm' | 'md' | 'lg';
}
```

| Padding | 클래스 |
|---------|--------|
| `none` | 없음 |
| `sm` | `p-3` |
| `md` | `p-4` (기본) |
| `lg` | `p-6` |

공통: `rounded-xl border border-border-default bg-surface`. className passthrough 지원.

### Badge

```typescript
interface BadgeProps extends ComponentPropsWithoutRef<'span'> {
  variant?: 'default' | 'info' | 'warning' | 'critical' | 'success';
}
```

`<span>` 기반, `inline-flex rounded-full px-2.5 py-0.5 text-xs font-medium`. 시맨틱 컬러 팔레트 적용.

헬퍼: `SeverityBadge({ severity })` — severity 문자열을 variant로 자동 매핑.

### Input

```typescript
// forwardRef 지원
interface InputProps extends InputHTMLAttributes<HTMLInputElement> {
  label?: string;
  error?: string;
}
```

- `label`: 상단 라벨 (`text-sm font-medium`)
- `error`: 하단 에러 메시지 (`text-red-400`), border도 red로 변경
- `block w-full rounded-lg`, focus 시 `ring-blue-500`
- disabled 시 `bg-surface-hover text-muted`

### Modal

```typescript
interface ModalProps {
  isOpen: boolean;
  onClose: () => void;
  title: string;
  children: ReactNode;
}
```

- 오버레이: `fixed inset-0 bg-black/70 z-50`
- 다이얼로그: `max-w-lg bg-surface border rounded-xl z-10`
- ESC 키, 외부 클릭으로 닫기
- ARIA: `role="dialog"`, `aria-modal="true"`

### Tooltip

```typescript
interface TooltipProps {
  content: ReactNode;
  position?: 'top' | 'bottom';
  align?: 'left' | 'center' | 'right';
  width?: string;           // 기본: w-56
  children?: ReactNode;     // 없으면 InfoIcon 렌더
}
```

- `group-hover`/`group-focus-within`으로 표시
- `backdrop-blur`, `rounded-lg border`, `text-xs leading-relaxed`
- `z-50`, opacity 트랜지션

### Avatar

```typescript
interface AvatarProps {
  src?: string | null;
  name: string;
  size?: 'sm' | 'md' | 'lg';
}
```

| Size | 클래스 |
|------|--------|
| `sm` | `h-6 w-6 text-xs` |
| `md` | `h-8 w-8 text-sm` (기본) |
| `lg` | `h-10 w-10 text-base` |

- `src` 있으면 `<img>` (rounded-full, object-cover)
- 없으면 이니셜 표시 (blue-500/15 배경, blue-400 텍스트)

### Spinner

```typescript
interface SpinnerProps {
  size?: 'sm' | 'md' | 'lg';
}
```

| Size | 클래스 |
|------|--------|
| `sm` | `h-4 w-4` |
| `md` | `h-6 w-6` (기본) |
| `lg` | `h-8 w-8` |

SVG 기반, `animate-spin`, `currentColor` 사용.

### Icons

`icons.tsx`에서 export. 베이스 `Icon` 컴포넌트가 `size` prop (기본 20px) 제공.

**제공 아이콘:** `CalendarIcon`, `TrendingUpIcon`, `WarningIcon`, `UsersIcon`, `CheckCircleIcon`, `ZapIcon`, `DollarIcon`, `CpuIcon`, `FolderIcon`, `InfoIcon`, `FileIcon`

모두 24x24 viewBox, stroke 기반 (Feather 스타일), `currentColor`.

### ErrorBoundary

```typescript
interface ErrorBoundaryProps {
  children: ReactNode;
  fallback?: ReactNode;
}
```

React class component. 에러 발생 시 경고 아이콘 + 에러 메시지 + "Try Again"/"Reload Page" 버튼 표시. 커스텀 `fallback` prop 지원.

### ApiErrorBoundary (ErrorDisplay)

```typescript
interface ErrorDisplayProps {
  message: string;
  onRetry?: () => void;
}
```

API 에러 표시 컴포넌트. `red-500/30` border, `red-500/10` 배경, 선택적 retry 버튼.

## 5. 테마 시스템

### 흐름

```
Zustand (useThemeStore)
  → localStorage ('context-sync-theme')
  → useThemeSync() hook
    → document.documentElement.setAttribute('data-theme', theme)
      → CSS custom properties 전환
```

### 구현

**Store** (`stores/theme.store.ts`):
```typescript
type Theme = 'dark' | 'light';

interface ThemeState {
  readonly theme: Theme;
  toggleTheme: () => void;
}
```
- Zustand `persist` 미들웨어로 localStorage 영속
- 기본값: `dark`

**Hook** (`hooks/use-theme.ts`):
```typescript
export function useThemeSync() {
  const theme = useThemeStore((s) => s.theme);
  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme);
  }, [theme]);
}
```
- `App.tsx`에서 호출하여 DOM 속성 동기화

**Toggle** (`Header.tsx` 내 ThemeToggle):
- Dark 모드: 태양 아이콘 (클릭 시 Light로)
- Light 모드: 달 아이콘 (클릭 시 Dark로)

## 6. 컴포넌트 패턴

### Record 기반 Variant 매핑

```typescript
const variantStyles: Record<Variant, string> = {
  primary: '...',
  secondary: '...',
  danger: '...',
  ghost: '...',
};

// 사용
className={`${baseStyles} ${variantStyles[variant]} ${sizeStyles[size]} ${className}`}
```

모든 variant/size 컴포넌트에 일관 적용.

### forwardRef

폼 요소(Input 등)는 `forwardRef`로 ref 전달 지원:

```typescript
export const Input = forwardRef<HTMLInputElement, InputProps>(
  ({ label, error, className, ...props }, ref) => { ... }
);
```

### className Passthrough

모든 컴포넌트가 외부 `className`을 수용하여 스타일 확장 가능:

```typescript
<Card className="mt-4" padding="lg">...</Card>
```

### HTML Attribute Spread

`...props` 패턴으로 네이티브 HTML 속성 전달:

```typescript
<button {...props} className={computedClassName}>
```

## 8. 접근성 & 인터랙션

- **Focus ring:** `focus:ring-2 focus:ring-offset-2 focus:ring-offset-page`
- **Hover 트랜지션:** `transition-colors` (대부분의 인터랙티브 요소)
- **ARIA:** Modal(`role="dialog"`, `aria-modal`), Tooltip(`role="tooltip"`), 버튼(`aria-label`)
- **키보드:** Modal ESC 닫기
- **Z-Index:** Modal/Tooltip `z-50`
